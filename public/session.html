Spotify Session

Manage your Spotify connection

Review session status, see expiry times, and reconnect if needed. Guests can view status but only the host can connect or disconnect.

Connect

This app uses a small backend so your Spotify credentials never touch the browser.

Not connected

Set `SPOTIFY_CLIENT_ID` and `SPOTIFY_CLIENT_SECRET` on the server, then run `node server.js`.
